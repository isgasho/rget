# rget

## Installation
```bash
go get github.com/orisano/rget
```

## How to use
```
$ rget
Usage of rget:
  -P int
    	maxprocs (default 4)
  -b int
    	block size (MB)
  -o string
    	output file path (required)
  -u string
    	url (required)
  -v	show verbose
  -x	add executable flag
```
